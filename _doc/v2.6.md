```rust
Archive {
    #[arg(long = "format")]
    format: Option<String>,
    tree: Option<String>,
}
```

在 `src/commands.rs` 新增 `archive()` 函式：
- 支援 `git5 archive --format=zip HEAD` - 產生 archive

### 2. 新增 blame 指令

在 `src/main.rs` 新增：
```rust
Blame {
    file: String,
}
```

在 `src/commands.rs` 新增 `blame()` 函式：
- 支援 `git5 blame <file>` - 顯示檔案每行的作者資訊
