### 1. 新增 describe 指令

在 `src/main.rs` 新增：
```rust
Describe {
    #[arg(long = "tags")]
    tags: bool,
    #[arg(long = "abbrev", default_value = "7")]
    abbrev: u32,
}
```

在 `src/commands.rs` 新增 `describe()` 函式：
- 支援 `git5 describe` - 描述目前 commit
- 支援 `git5 describe --tags` - 包含標籤
- 支援 `git5 describe --abbrev=7` - 指定簡短長度
