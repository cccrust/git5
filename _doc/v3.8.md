## 開發摘要

本版本實作 format-patch 和 send-email 指令。

## 實作細節

### 1. 新增 format-patch 指令

在 `src/main.rs` 新增：
```rust
FormatPatch {
    since: Option<String>,
},
```

在 `src/commands.rs` 新增 `format_patch()` 函式：
- 支援 `git5 format-patch HEAD~1` - 產生 patch 檔案
- 支援 `git5 format-patch` - 產生所有最新 commit 的 patch

### 2. 新增 send-email 指令

在 `src/main.rs` 新增：
```rust
SendEmail {
    patch: String,
},
```

在 `src/commands.rs` 新增 `send_email()` 函式：
- 支援 `git5 send-email <patch>` - 發送 patch（模擬）
