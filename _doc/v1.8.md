## 開發摘要

本版本實作 show-ref 和 count-objects 指令。

## 實作細節

### 1. 新增 show-ref 指令

在 `src/main.rs` 新增：
```rust
ShowRef {
    #[arg(long = "heads")]
    heads: bool,
    #[arg(long = "tags")]
    tags: bool,
}
```

在 `src/commands.rs` 新增 `show_ref()` 函式：
- 支援 `git5 show-ref` - 列出所有參照
- 支援 `git5 show-ref --heads` - 僅列出分支
- 支援 `git5 show-ref --tags` - 僅列出標籤

### 2. 新增 count-objects 指令

在 `src/main.rs` 新增：
```rust
CountObjects {
    #[arg(long = "verbose")]
    verbose: bool,
}
```

在 `src/commands.rs` 新增 `count_objects()` 函式：
- 支援 `git5 count-objects` - 統計物件數量
- 支援 `git5 count-objects --verbose` - 顯示詳細資訊

### 3. 新增整合測試

- `test_show_ref` - 測試 show-ref
- `test_count_objects` - 測試 count-objects
