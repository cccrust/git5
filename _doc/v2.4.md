## 開發摘要

本版本實作 cat-file --batch 和 diff-tree 指令。

## 實作細節

### 1. 新增 cat-file --batch 指令

在 `src/main.rs` 新增：
```rust
CatFileBatch {
    #[arg(short)]
    batch: bool,
}
```

在 `src/commands.rs` 新增 `cat_file_batch()` 函式。

### 2. 新增 diff-tree 指令

在 `src/main.rs` 新增：
```rust
DiffTree {
    tree1: String,
    tree2: Option<String>,
}
```

在 `src/commands.rs` 新增 `diff_tree()` 和 `parse_tree_content()` 函式：
- 支援 `git5 diff-tree <tree1>` - 顯示 tree 內容
- 支援 `git5 diff-tree <tree1> <tree2>` - 比較兩 tree 差異
