## 開發摘要

本版本實作 fsck 和 gc 指令。

## 實作細節

### 1. 新增 fsck 指令

在 `src/main.rs` 新增：
```rust
Fsck {
    verbose: bool,
},
```

在 `src/commands.rs` 新增 `fsck()` 函式：
- 支援 `git5 fsck` - 驗證 repository 完整性
- 支援 `git5 fsck --verbose` - 顯示詳細錯誤

### 2. 新增 gc 指令

在 `src/main.rs` 新增：
```rust
Gc {
    aggressive: bool,
},
```

在 `src/commands.rs` 新增 `gc()`、`get_reachable_objects()`、`collect_objects()`、`collect_tree_objects()` 函式：
- 支援 `git5 gc` - 垃圾回收
- 支援 `git5 gc --aggressive` - 積極回收
