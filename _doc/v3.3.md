**狀態**: 完成

## 開發摘要

本版本實作 fetch --all 和 reset 指令。

## 實作細節

### 1. 新增 fetch --all 指令

在 `src/main.rs` 新增：
```rust
FetchAll {
    #[arg(long)]
    all: bool,
},
```

在 `src/commands.rs` 新增 `fetch_all()` 函式：
- 支援 `git5 fetch --all` - 從所有 remote fetch

### 2. 新增 reset 指令

在 `src/main.rs` 新增：
```rust
Reset {
    #[arg(long)]
    hard: bool,
    #[arg(long)]
    soft: bool,
    commit: Option<String>,
},
```

在 `src/commands.rs` 新增 `reset()` 函式：
- 支援 `git5 reset --hard <commit>` - 重置到指定 commit 並重置工作目錄
- 支援 `git5 reset --soft <commit>` - 軟重置
- 支援 `git5 reset <commit>` - 預設重置
