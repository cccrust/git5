## 開發摘要

本版本實作 bisect 和 worktree 指令。

## 實作細節

### 1. 新增 bisect 指令

在 `src/main.rs` 新增：
```rust
Bisect {
    subcommand: String,
    commit: Option<String>,
},
```

在 `src/commands.rs` 新增 `bisect()`、`run_bisect()`、`get_commit_halfway()` 函式：
- 支援 `git5 bisect start` - 開始 bisect
- 支援 `git5 bisect good <commit>` - 標記為 good
- 支援 `git5 bisect bad <commit>` - 標記為 bad
- 支援 `git5 bisect reset` - 重置 bisect

### 2. 新增 worktree 指令

在 `src/main.rs` 新增：
```rust
Worktree {
    subcommand: String,
    path: Option<String>,
    branch: Option<String>,
},
```

在 `src/commands.rs` 新增 `worktree()` 函式：
- 支援 `git5 worktree add <path> <branch>` - 新增 worktree
- 支援 `git5 worktree list` - 列出 worktree
- 支援 `git5 worktree remove <path>` - 移除 worktree
