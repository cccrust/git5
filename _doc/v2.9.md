## 開發摘要

本版本實作 stash 和 cherry-pick 指令。

## 實作細節

### 1. 新增 stash 指令

在 `src/main.rs` 新增：
```rust
Stash {
    #[arg(short, long, default_value = "")]
    message: String,
    list: bool,
    pop: bool,
},
```

在 `src/commands.rs` 新增 `stash()` 函式：
- 支援 `git5 stash save "message"` - 儲存工作目錄變更
- 支援 `git5 stash list` - 列出 stash
- 支援 `git5 stash pop` - 恢復 stash

### 2. 新增 cherry-pick 指令

在 `src/main.rs` 新增：
```rust
CherryPick {
    commit: String,
},
```

在 `src/commands.rs` 新增 `cherry_pick()` 函式：
- 支援 `git5 cherry-pick <commit>` - 撿取指定 commit
