## 開發摘要

本版本實作 ls-tree 和 update-ref 指令。

## 實作細節

### 1. 新增 ls-tree 指令

在 `src/main.rs` 新增：
```rust
LsTree {
    #[arg(short)]
    recursive: bool,
    tree: String,
}
```

在 `src/commands.rs` 新增 `ls_tree()` 函式：
- 支援 `git5 ls-tree <tree>` - 列出 tree 內容
- 支援 `git5 ls-tree -r <tree>` - 遞迴列出

### 2. 新增 update-ref 指令

在 `src/main.rs` 新增：
```rust
UpdateRef {
    #[arg(short)]
    delete: bool,
    ref_name: String,
    hash: Option<String>,
}
```

在 `src/commands.rs` 新增 `update_ref()` 函式：
- 支援 `git5 update-ref refs/heads/main <hash>` - 更新參照
- 支援 `git5 update-ref -d refs/heads/main` - 刪除參照
