## 開發摘要

## 實作細節

### 更新 init 指令

在 `src/main.rs` 新增參數：
```rust
Init {
    #[arg(long)]
    bare: bool,
    path: Option<String>,
},
```

在 `src/commands.rs` 更新 `init()` 函式：
- 支援 `git5 init` - 一般模式，建立 .git4/
- 支援 `git5 init --bare` - 在當前目錄建立 bare repo
- 支援 `git5 init --bare myrepo` - 建立指定名稱的 bare repo
- 支援 `git5 init myrepo` - 在子目錄建立 .git4/
