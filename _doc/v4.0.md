## 開發摘要

本版本是里程碑版本，實作 diff --stat 指令。

## 實作細節

### 1. 更新 diff 指令

在 `src/main.rs` 更新：
```rust
Diff {
    file: String,
    #[arg(long = "stat")]
    stat: bool,
},
```

在 `src/commands.rs` 更新 `diff()` 函式：
- 支援 `git5 diff <file>` - 顯示詳細差異
- 支援 `git5 diff --stat <file>` - 顯示統計資訊
