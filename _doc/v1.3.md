# git5 - 開發紀錄 v1.3

**日期**: 2026-05-04
**版本**: v1.3.0
**狀態**: 完成

## 開發摘要

本版本新增 `config` 指令，並擴展整合測試覆蓋範圍。

## 實作細節

### 1. 新增 config 指令

在 `src/main.rs` 新增 `Config` 命令定義：
```rust
Config {
    #[arg(long = "list")]
    list: bool,
    key: Option<String>,
    value: Option<String>,
}
```

在 `src/commands.rs` 新增：
- `config()` 函式 - 處理 config 指令
- `parse_config()` 函式 - 解析 config 檔案
- `save_config()` 函式 - 儲存 config 檔案

支援功能：
- `git5 config <key> <value>` - 設定值
- `git5 config <key>` - 取得值
- `git5 config --list` - 列出所有設定

### 2. 擴展整合測試

在 `tests/integration.rs` 新增 4 個測試：
- `test_branch` - 分支建立與列表
- `test_config_set_get` - config 設定與取得
- `test_config_list` - config 列表
- `test_log` - 檢視 commit 歷史
