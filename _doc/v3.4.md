# git5 - 開發紀錄 v3.4

**日期**: 2026-05-04
**版本**: v3.4.0
**狀態**: 完成

## 開發摘要

本版本實作 apply 和 switch 指令。

## 實作細節

### 1. 新增 apply 指令

在 `src/main.rs` 新增：
```rust
Apply {
    #[arg(long)]
    check: bool,
    patch: String,
},
```

在 `src/commands.rs` 新增 `apply()` 函式：
- 支援 `git5 apply <patch>` - 套用 patch
- 支援 `git5 apply --check <patch>` - 檢查 patch 是否可套用

### 2. 新增 switch 指令

在 `src/main.rs` 新增：
```rust
Switch {
    create: bool,
    branch: String,
},
```

在 `src/commands.rs` 新增 `switch()` 函式：
- 支援 `git5 switch <branch>` - 切換分支
- 支援 `git5 switch -c <branch>` - 建立並切換
