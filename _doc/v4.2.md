## 開發摘要

## 實作細節

### 1. 新增 ls-files --cached 指令

在 `src/main.rs` 新增：
```rust
LsFilesCached {
    #[arg(long = "cached")]
    cached: bool,
},
```

在 `src/commands.rs` 新增 `ls_files_cached()` 函式：
- 支援 `git5 ls-files --cached` - 顯示 staged 檔案

### 2. 新增 rev-parse --abbrev-ref 指令

在 `src/main.rs` 新增：
```rust
RevParseAbbrev {
    #[arg(long = "abbrev-ref")]
    abbrev_ref: bool,
},
```

在 `src/commands.rs` 新增 `rev_parse_abbrev()` 函式：
- 支援 `git5 rev-parse --abbrev-ref HEAD` - 顯示目前分支簡稱
