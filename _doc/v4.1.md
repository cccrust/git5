# git5 - 開發紀錄 v4.1

**日期**: 2026-05-04
**版本**: v4.1.0
**狀態**: 完成

## 開發摘要

本版本實作 commit --amend 和 help 指令。

## 實作細節

### 1. 新增 commit --amend 指令

在 `src/main.rs` 新增：
```rust
CommitAmend {
    #[arg(long = "amend")]
    amend: bool,
    message: Option<String>,
},
```

在 `src/commands.rs` 新增 `commit_amend()` 函式：
- 支援 `git5 commit --amend` - 修改最後一次 commit
- 支援 `git5 commit --amend -m <message>` - 修改最後一次 commit 並更新訊息

### 2. 新增 help 指令

在 `src/main.rs` 新增：
```rust
GitHelp {
    command: Option<String>,
},
```

在 `src/commands.rs` 新增 `help()` 函式：
- 支援 `git5 help` - 顯示指令列表
- 支援 `git5 help <command>` - 顯示特定指令說明
