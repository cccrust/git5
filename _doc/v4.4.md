## 實作內容

### 1. pull 指令 (fetch + merge)
- 新增 `Command::Pull` variant
- fetch: 複製 remote 的 refs 和 objects
- merge: 自動 checkout remote 的 commit 到 working directory
- CLI: `git5 pull <remote_path>`

### 2. restore 指令
- 從 HEAD 還原檔案到 working directory
- 支援 `--staged` 從 index 還原
- CLI: `git5 restore [--staged] <path>`

### 3. show 指令
- 顯示 commit/tree/blob 內容
- 預設顯示 HEAD，也可以指定 revision
- CLI: `git5 show [revision]`

### 4. 修復 fetch
- fetch 原本只是 stub，現在會複製 remote objects 和 refs
