## 開發摘要

本版本實作 symbolic-ref 和 for-each-ref 指令。

## 實作細節

### 1. 新增 symbolic-ref 指令

在 `src/main.rs` 新增：
```rust
SymbolicRef {
    ref_name: Option<String>,
    target: Option<String>,
}
```

在 `src/commands.rs` 新增 `symbolic_ref()` 函式：
- 支援 `git5 symbolic-ref HEAD` - 讀取 symbolic ref
- 支援 `git5 symbolic-ref HEAD refs/heads/main` - 設定 symbolic ref

### 2. 新增 for-each-ref 指令

在 `src/main.rs` 新增：
```rust
ForEachRef {
    #[arg(long = "format")]
    format: Option<String>,
}
```

在 `src/commands.rs` 新增 `for_each_ref()` 函式：
- 支援 `git5 for-each-ref` - 列出所有 ref
