## 開發摘要

本版本實作 log --oneline 和 status --short 指令。

## 實作細節

### 1. 新增 log --oneline 指令

在 `src/main.rs` 新增：
```rust
LogOneline {
    #[arg(long = "oneline")]
    oneline: bool,
},
```

在 `src/commands.rs` 新增 `log_oneline()` 函式：
- 支援 `git5 log --oneline` - 顯示精簡 commit 歷史

### 2. 新增 status --short 指令

在 `src/main.rs` 新增：
```rust
StatusShort {
    #[arg(long = "short")]
    short: bool,
},
```

在 `src/commands.rs` 新增 `status_short()` 函式：
- 支援 `git5 status --short` - 顯示精簡工作目錄狀態
