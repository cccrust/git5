# git5 - 開發紀錄 v1.6

**日期**: 2026-05-04
**版本**: v1.6.0
**狀態**: 完成

## 開發摘要

本版本實作 tag 和 rm 指令，新增相關整合測試。

## 實作細節

### 1. 新增 tag 指令

在 `src/main.rs` 新增：
```rust
Tag {
    #[arg(short)]
    delete: bool,
    name: Option<String>,
}
```

在 `src/commands.rs` 新增：
- `tag()` 函式 - 處理 tag 指令
- 支援 `git5 tag` - 列出標籤
- 支援 `git5 tag <name>` - 建立輕量標籤
- 支援 `git5 tag -d <name>` - 刪除標籤

### 2. 新增 rm 指令

在 `src/main.rs` 新增：
```rust
Rm {
    files: Vec<String>,
}
```

在 `src/commands.rs` 新增：
- `rm()` 函式 - 從 index 移除檔案
- 支援 `git5 rm <file>` - 移除檔案

### 3. 新增整合測試

在 `tests/integration.rs` 新增：
- `test_tag` - 測試建立標籤
- `test_tag_delete` - 測試刪除標籤
