## 開發摘要

v2.0 是重要里程碑版本，代表 git5 達到穩定的里程碑。本版本專注於測試驗證與程式碼品質。

## 實作成果

### 支援的指令

**基礎指令**:
- `init` - 初始化儲存庫
- `hash-object` - 計算物件 hash
- `cat-file` - 顯示物件內容
- `write-tree` - 建立 tree 物件
- `commit-tree` - 建立 commit 物件

**版本控制**:
- `add` - 加入檔案到 index
- `commit` - 建立 commit
- `log` - 顯示 commit 歷史
- `status` - 顯示工作目錄狀態
- `diff` - 顯示差異

**分支管理**:
- `branch` - 列出/建立分支
- `checkout` - 切換分支（含 -b 建立新分支）
- `merge` - 合併分支

**遠端操作**:
- `clone` - 複製儲存庫
- `push` - 推送到遠端
- `fetch` - 從遠端擷取
- `remote` - 管理遠端
- `ls-remote` - 列出遠端參照

**設定**:
- `config` - 設定管理
- `tag` - 標籤管理
- `ls-files` - 列出 index 檔案
- `rev-parse` - 解析 revision
- `show-ref` - 列出參照
- `count-objects` - 統計物件
- `describe` - 描述目前 commit
- `rm` - 移除檔案

### 模組結構

```
src/
├── lib.rs         - Library 入口
├── main.rs        - CLI 入口
├── error.rs       - 錯誤類型定義
├── object.rs      - 物件操作
├── tree.rs        - Tree 處理
├── commit.rs      - Commit 處理
├── index.rs       - Index 管理
└── commands.rs    - 命令實作
```
