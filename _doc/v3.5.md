## 開發摘要

本版本實作 fetch --prune 和 reflog 指令。

## 實作細節

### 1. 新增 fetch --prune 指令

在 `src/main.rs` 新增：
```rust
FetchPrune {
    #[arg(long)]
    prune: bool,
},
```

在 `src/commands.rs` 新增 `fetch_prune()` 函式：
- 支援 `git5 fetch --prune` - fetch 並清理 remote ref

### 2. 新增 reflog 指令

在 `src/main.rs` 新增：
```rust
Reflog {
    subcommand: String,
},
```

在 `src/commands.rs` 新增 `reflog()` 函式：
- 支援 `git5 reflog show` - 顯示 reflog
- 支援 `git5 reflog expire` - 過期 reflog
