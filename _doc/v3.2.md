## 開發摘要

本版本實作 submodule 和 clean 指令。

## 實作細節

### 1. 新增 submodule 指令

在 `src/main.rs` 新增：
```rust
Submodule {
    subcommand: String,
    url: Option<String>,
    path: Option<String>,
},
```

在 `src/commands.rs` 新增 `submodule()` 函式：
- 支援 `git5 submodule add <url> <path>` - 新增 submodule
- 支援 `git5 submodule list` - 列出 submodule
- 支援 `git5 submodule update` - 更新 submodule

### 2. 新增 clean 指令

在 `src/main.rs` 新增：
```rust
Clean {
    force: bool,
    directories: bool,
},
```

在 `src/commands.rs` 新增 `clean()` 函式：
- 支援 `git5 clean` - 預覽要移除的檔案
- 支援 `git5 clean -f` - 移除未追蹤檔案
- 支援 `git5 clean -fd` - 移除未追蹤檔案和目錄
