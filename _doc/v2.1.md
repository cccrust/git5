## 開發摘要

本版本擴展 cat-file 指令並新增 verify-pack 和 mktree 指令。

## 實作細節

### 1. 擴展 cat-file 指令

在 `src/main.rs` 修改：
```rust
CatFile {
    #[arg(short = 'p')]
    print: bool,
    #[arg(short = 's')]
    size: bool,
    #[arg(short = 't')]
    type_flag: bool,
    object: String,
}
```

在 `src/commands.rs` 新增：
- `git5 cat-file -s <object>` - 顯示物件大小
- `git5 cat-file -t <object>` - 顯示物件類型

### 2. 新增 verify-pack 指令

在 `src/commands.rs` 新增 `verify_pack()` 函式：
- 驗證 pack 檔案的 header
- 檢查 magic number 和 version
- 顯示物件數量

### 3. 新增 mktree 指令

在 `src/commands.rs` 新增 `mktree()` 函式：
- 從 index 建立 tree 物件
